# Introduction

Welcome to the **StructUI** documentation!

StructUI is a comprehensive framework designed to bridge the gap between complex data representations and intuitive user interfaces. It provides a unified parser and UI generation system that dynamically builds editable, validation-backed GUIs based on underlying data schemas.

## Why StructUI?
- **Unified AST**: Parse diverse formats (JSON, YAML, CSV, raw binary) into a single Abstract Syntax Tree.
- **Dynamic UI Generation**: Automatically generate NiceGUI-based interfaces that reflect the schema structure.
- **Strict Validation**: Enforce data integrity through immutable validation rules applied at the node level.
- **Extensible Architecture**: Easily integrate new parsers without touching the core UI components.

## Target Audience
This documentation is intended for developers who want to:
- Learn the internal architecture of StructUI.
- Map custom parsers to the StructUI AST.
- Embed the StructUI designer within their own custom applications.
